cliForumSocket

[순수 JAVA 코드로 작성한 TCP 통신 게시판]


cliForumSocket은 순수 java를 이용해 개발한 tcp 통신 게시판입니다.

회원가입, 글 쓰기, 댓글 달기까지 게시판의 기본에 충실하면서 통신 기능까지 탑재해서 동시에 여러명이 접속해서 사용 할 수 있도록 개발하였습니다.

게시글이나 댓글이 달렸을 때 실시간으로 화면이 갱신되는 기능까지는 구현을 실패했고, 대신 새로고침 버튼을 누르면 최신 데이터를 읽어 올 수 있도록 만들었습니다.

Two-Ex

[전문기술 쇼핑 플랫폼과 학습관리시스템의 합체]


투엑스는 서비스 쇼핑 플랫폼과 학습관리시스템의 조합입니다.

전문가 회원은 교육, 트레이닝 등의 서비스를 상품으로 사이트에 등록하고 제공 할 수 있습니다.

고객회원은 상품을 구매하고 학습관리시스템에 참여 할 수 있습니다.

학습관리시스템을 통해서, 고객은 전문가, 또는 기타 고객들과의 교류가 가능합니다.
